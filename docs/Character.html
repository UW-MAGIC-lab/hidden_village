height: Number (in pixels); often a scalar value of the height or other segment length

facePosition: Array; An array with an x and y value, with (0,0) being the top left corner of the character

mood: String; One of ["happy", "sad", "angry", "neutral", "surprised"]

placement: Array; An array with an x and y value, with (0,0) being the top left corner of the screen
